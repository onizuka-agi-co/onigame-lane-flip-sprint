## コンセプト

Lane Flip Sprint は 1画面完結の短時間サバイバルです。

- 3つの固定レーン
- 左右だけのレーン移動
- 45秒生存でクリア
- 失敗条件は被弾1つ

## ローカル実行

```bash
# ビルド不要
# index.html をブラウザで開くだけ
```

## 操作

- `Leftボタン` / `ArrowLeft` / `A`: 左に移動
- `Rightボタン` / `ArrowRight` / `D`: 右に移動
- `Retry` または `Space`: リトライ
